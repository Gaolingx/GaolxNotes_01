# FID与KID区别

### 🧑‍💻 User

图像指标的kid和fid有什么区别？

---

### 🤖 Assistant

**FID (Fréchet Inception Distance)** 和 **KID (Kernel Inception Distance)** 都是计算机视觉领域中用于评估生成模型（如 GANs、扩散模型）生成图像质量和多样性的核心指标。它们的基本思想都是利用预训练模型（通常是 Inception-v3）提取真实图像和生成图像的特征，然后计算这两个特征分布之间的距离。距离越小，说明生成的图像越逼真、质量越高。

尽管目的相同，但它们在**底层统计学原理、对样本数量的敏感度以及计算方式**上有着显著的区别。

以下是两者的详细区别：

### 1. 底层统计学原理假设

*   **FID（参数化方法）：**
    FID 假设提取出的图像特征服从**多元高斯分布（正态分布）**。它计算的是真实图像特征分布 $\mathcal{N}(\mu_r, \Sigma_r)$ 和生成图像特征分布 $\mathcal{N}(\mu_g, \Sigma_g)$ 之间的 Fréchet 距离（也称为 Wasserstein-2 距离）。
    公式中依赖于均值向量 $\mu$ 和协方差矩阵 $\Sigma$。如果真实的特征分布并不是高斯分布，FID 的评估结果就会出现偏差。
*   **KID（非参数化方法）：**
    KID **不假设特征服从任何特定的概率分布**（不需要高斯假设）。它计算的是真实图像和生成图像特征之间的**最大均值差异（Maximum Mean Discrepancy, MMD）**的平方。KID 通常使用多项式核函数（Polynomial Kernel，如三次多项式）来测量两个分布在再生核希尔伯特空间（RKHS）中的距离。

### 2. 对样本数量的依赖（有偏 vs 无偏）

这是两者在实际应用中最关键的区别：

*   **FID 是“有偏估计”（Biased Estimator）：**
    FID 的计算结果高度依赖于测试样本的数量。样本量越小，FID 的计算误差越大（通常会高估距离）。为了获得稳定且可比较的 FID 分数，学术界通常需要使用大量的图像（标准的做法是使用 **50,000** 张图像）。如果你只用 1,000 张图像算出的 FID 和别人用 50,000 张算出的 FID 是**完全没有可比性**的。
*   **KID 是“无偏估计”（Unbiased Estimator）：**
    KID 的期望值不随样本数量的变化而变化。这意味着即使你的样本量很少（例如只有几百或一两千张图像），KID 依然能够给出一个相对稳定和准确的评估分数。KID 内部通常使用分块估计（Block Estimator）来计算，这进一步提高了它在小样本下的稳定性。

### 3. 计算复杂度

*   **FID：** 计算协方差矩阵的平方根（Matrix Square Root），时间复杂度大致为 $O(d^3)$，其中 $d$ 是特征维度（Inception 特征通常是 2048 维）。在样本量很大时，提取特征是主要开销，距离计算相对较快。
*   **KID：** 计算 MMD 需要计算核矩阵（Kernel Matrix），传统 MMD 计算的复杂度与样本量 $N$ 的平方成正比 $O(N^2)$。为了解决这个问题，KID 采用分块计算（将数据分成多个小的 batch 计算然后求平均），从而在计算效率和内存消耗上达到平衡。

---

### 总结与对比表

| 特性 | FID (Fréchet Inception Distance) | KID (Kernel Inception Distance) |
| :--- | :--- | :--- |
| **分布假设** | 假设特征服从多元高斯分布 | 无分布假设（非参数化） |
| **距离度量** | Fréchet / Wasserstein-2 距离 | 基于核函数的最大均值差异 (MMD) |
| **样本量敏感度** | **高**（有偏估计） | **低**（无偏估计） |
| **所需样本量** | 通常需要很大（推荐 50k） | 小样本下依然稳定可靠 |
| **行业地位** | **绝对的主流标准**，论文必备 | 逐渐流行的补充指标 |

1.  **如果你在写论文或做标准评测：** **必须报告 FID**。因为 FID 是目前生成领域的“黄金标准”，只有提供 FID 分数，你的结果才能与过去几年发布的绝大多数论文进行公平对比。记得确保你的样本量（通常是 50k）与对比的 Baseline 一致。
2.  **如果你的数据集很小（例如只有几百、几千张图片）：** **首选 KID**。在小样本下，FID 的分数会非常不稳定且缺乏意义，而 KID 的无偏特性使得它依然能够准确反映模型的生成质量。
3.  **最佳实践：** 目前越来越多的高质量论文会**同时报告 FID 和 KID**，以此来全面证明模型不仅在大规模生成下表现优异，而且在特征分布匹配上没有受到高斯假设的局限。
