# 代码引用英文注释写法

### 🧑‍💻 User

代码是从其他github仓库拷贝的要如何写英文注释？
```
@inproceedings{wolf-etal-2020-transformers,
    title = "Transformers: State-of-the-Art Natural Language Processing",
    author = "Thomas Wolf and Lysandre Debut and Victor Sanh and Julien Chaumond and Clement Delangue and Anthony Moi and Pierric Cistac and Tim Rault and Rémi Louf and Morgan Funtowicz and Joe Davison and Sam Shleifer and Patrick von Platen and Clara Ma and Yacine Jernite and Julien Plu and Canwen Xu and Teven Le Scao and Sylvain Gugger and Mariama Drame and Quentin Lhoest and Alexander M.     year = "2020",

    url = "https://www.aclweb.org/anthology/2020.emnlp-demos.6",
```

### 🤖 Assistant

在代码中注明出处是一个非常好的开源和学术习惯。针对你提供的这篇 Hugging Face Transformers 的参考文献（BibTeX），你可以根据需要的详细程度，选择以下几种英文注释的写法：

### 1. 详细版（推荐放在文件头部或核心类的文档字符串 Docstring 中）
如果你拷贝了很大一段代码，或者整个文件都是基于该仓库修改的，建议使用多行注释，并把完整的引用信息放进去：

```python
"""
This code is adapted from the Hugging Face Transformers repository.
Original source: https://github.com/huggingface/transformers
(Optional: add the specific file URL here)

Reference:
@inproceedings{wolf-etal-2020-transformers,
    title = "Transformers: State-of-the-Art Natural Language Processing",
    author = "Thomas Wolf and Lysandre Debut and Victor Sanh and Julien Chaumond and Clement Delangue and Anthony Moi and Pierric Cistac and Tim Rault and Rémi Louf and Morgan Funtowicz and Joe Davison and Sam Shleifer and Patrick von Platen and Clara Ma and Yacine Jernite and Julien Plu and Canwen Xu and Teven Le Scao and Sylvain Gugger and Mariama Drame and Quentin Lhoest and Alexander M. Rush",
    booktitle = "Proceedings of the 2020 Conference on Empirical Methods in Natural Language Processing: System Demonstrations",
    month = oct,
    year = "2020",
    publisher = "Association for Computational Linguistics",
    url = "https://www.aclweb.org/anthology/2020.emnlp-demos.6",
    pages = "38--45"
}
"""
```

### 2. 简洁版（适合放在函数或代码块上方）
如果你只拷贝了一个函数或一小段逻辑，可以直接用单行注释，写明出处和作者年份即可：

```python
# The following code block is adapted from Hugging Face Transformers (Wolf et al., 2020).
# Source: https://github.com/huggingface/transformers
def your_copied_function():
    ...
```

或者：

```python
# Acknowledgment: This implementation is based on the Hugging Face Transformers library.
# See: Wolf et al., "Transformers: State-of-the-Art Natural Language Processing" (2020).
```

### 3. 如果你对代码进行了修改（Modified）
通常我们拷贝代码后会根据自己的项目进行调整，这时可以使用 `modified from` 或 `adapted from` 来表达：

```python
# This function is heavily inspired by / adapted from the Hugging Face Transformers repository.
# Reference: Wolf et al. (2020), https://www.aclweb.org/anthology/2020.emnlp-demos.6
```

<details>
<summary>💡 <b>附加建议（点击展开）</b></summary>

*   **附上具体链接**：除了论文的 BibTeX，强烈建议在注释中附上你拷贝的那个代码文件的 **GitHub URL**（最好是带有 Commit Hash 的永久链接），这样方便以后追溯和对比。
*   **注意开源协议（License）**：Hugging Face Transformers 使用的是 Apache 2.0 协议。如果你拷贝了大量的核心代码，最好在你的项目中保留原作者的 License 声明，或者在文件头加上类似 `Copyright 2020 The HuggingFace Team. All rights reserved.` 的字样以规避版权风险。
</details>
