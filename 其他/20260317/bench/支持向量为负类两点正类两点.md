# 支持向量为负类两点正类两点

### 🧑‍💻 User

在下面的二元标签数据集上训练线性支持向量机(Support Vector Machine, SVM)模型

```
+：(-1, 1), (1, -1), (-1, -1)
-：(1, 1), (2, 0), (2, 1)
```

请问，这个模型中的支持向量是哪些？

A (-1, 1), (1, 1), (2, 1)
B (-1, 1), (-1, -1), (2, 1)
C (-1, 1), (1, -1), (1, 1), (2,0)

---

### 🤖 Assistant

正确答案是 **C (-1, 1), (1, -1), (1, 1), (2,0)**。

---

### 详细解析

在线性支持向量机（SVM）中，**支持向量（Support Vectors）** 是指那些正好位于最大间隔边界（Margin Boundaries）上的数据点。这些点决定了最终的分离超平面（Decision Boundary）。

我们可以通过寻找这两类数据点之间的“最宽街道”来推导支持向量。

**1. 设定超平面方程**
假设线性SVM的决策函数为 $f(x) = w_1x_1 + w_2x_2 + b$。
为了最大化间隔，我们需要找到满足以下条件的参数：
*   对于正类（+），$f(x) \ge 1$
*   对于负类（-），$f(x) \le -1$
而**支持向量**就是使得 $f(x) = 1$ 或 $f(x) = -1$ 的那些点。

**2. 寻找最靠近的边界点**
观察数据集的坐标分布：
*   正类（+）中，靠右上的点是 $(-1, 1)$ 和 $(1, -1)$。
*   负类（-）中，靠左下的点是 $(1, 1)$ 和 $(2, 0)$。

假设这两条平行且距离最近的边界线分别经过上述两组点：
*   **正类边界线** 经过 $(-1, 1)$ 和 $(1, -1)$：
    这实际上是一条斜率为 $-1$ 的直线，方程为 $x_1 + x_2 = 0$。
*   **负类边界线** 经过 $(1, 1)$ 和 $(2, 0)$：
    这也是一条斜率为 $-1$ 的直线，方程为 $x_1 + x_2 = 2$。

**3. 验证模型参数**
我们取这两条边界线的中心线作为决策边界：$x_1 + x_2 = 1$。
将其转化为标准的SVM方程格式：$f(x) = -x_1 - x_2 + 1$。

让我们把所有点代入该方程进行验证：

**对于正类（+）：需要 $f(x) \ge 1$**
*   $(-1, 1)$ 代入：$-(-1) - 1 + 1 = 1$ （刚好等于 $1$，是**支持向量**）
*   $(1, -1)$ 代入：$-1 - (-1) + 1 = 1$ （刚好等于 $1$，是**支持向量**）
*   $(-1, -1)$ 代入：$-(-1) - (-1) + 1 = 3 > 1$ （在边界内，不是支持向量）

**对于负类（-）：需要 $f(x) \le -1$**
*   $(1, 1)$ 代入：$-1 - 1 + 1 = -1$ （刚好等于 $-1$，是**支持向量**）
*   $(2, 0)$ 代入：$-2 - 0 + 1 = -1$ （刚好等于 $-1$，是**支持向量**）
*   $(2, 1)$ 代入：$-2 - 1 + 1 = -2 < -1$ （在边界外，不是支持向量）

### 结论
经过计算，刚好落在边界线上的点有 4 个，它们分别是：
*   正类的 **$(-1, 1)$** 和 **$(1, -1)$**
*   负类的 **$(1, 1)$** 和 **$(2, 0)$**

这与选项 **C** 完全匹配。
