# 多路划分导致过拟合

### 🧑‍💻 User

使用决策树分类时，如果输入的某个特征的值是连续的，通常使用二分法对连续属性离散化，即根据是否大于/小于某个阈值进行划分。如果采用多路划分，每个出现的值都划分为一个分支，这种方式的最大问题是：

A 计算量太大
B 训练集和测试集表现都很差
C 训练集表现良好，测试集表现差
D 训练集表现差，测试集表现良好

---

### 🤖 Assistant

正确答案是 **C 训练集表现良好，测试集表现差**。

### 原因分析：

如果对连续型特征采用**多路划分（即每一个出现过的具体数值都单独作为一个分支）**，会导致极其严重的**过拟合（Overfitting）**现象。

1.  **在训练集上（表现良好）：** 决策树会为训练集中出现的每一个具体数值建立一个分支。这相当于决策树直接把训练集的数据“死记硬背”了下来，把每一个样本都分到了一个纯度极高（甚至只有一个样本）的叶子节点中。因此，模型在训练集上的准确率会非常高，甚至达到 $100\%$。
2.  **在测试集上（表现差）：** 连续属性的取值空间是无限的。当面对测试集或真实场景中的新样本时，新样本的连续特征值极大概率与训练集中的任何一个具体数值都**不完全相等**（例如，训练集中有 $1.52$ 和 $1.54$，但测试集中出现了 $1.53$）。此时，决策树会找不到对应的分支来进行分类，导致模型完全失去泛化能力，测试集表现极差。

<details>
<summary><b>💡 扩展：为什么二分法（阈值划分）是更好的选择？</b></summary>

由于多路划分会带来严重的过拟合问题，C4.5 和 CART 等经典的决策树算法在处理连续属性时，都采用了**二分法（Bi-partition）**：

*   **泛化能力强：** 二分法通过寻找一个最佳阈值 $t$，将连续属性划分为 $A \le t$ 和 $A > t$ 两个区间（分支）。这种方式提取的是数据的“范围特征”而非“绝对值特征”，能够很好地覆盖未见过的新数据。
*   **计算方式：** 将训练集中该属性的所有取值从小到大排序，取相邻两个值的平均值作为候选阈值。然后计算每个候选阈值带来的信息增益（或基尼指数下降量），选择最优的阈值进行切分。
*   **特征复用：** 与离散特征不同，连续特征在二分法下，可以在决策树的后续子节点中被**重复使用**（即在不同的深度使用不同的阈值进行多次切分），这赋予了决策树更强的表达能力。

</details>
