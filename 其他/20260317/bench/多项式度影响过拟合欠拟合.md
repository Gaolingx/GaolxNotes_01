# 多项式度影响过拟合欠拟合

### 🧑‍💻 User

对多项式回归(polynomial regression)而言，哪一项对过拟合(overfitting)和欠拟合(underfitting)影响最大。

A 多项式的度(polynomial degree)
B 是否通过矩阵求逆/梯度下降学习权重
C 高斯噪声方差(variance of the Gaussian noise)
D 每一次训练的输入个数固定

---

### 🤖 Assistant

正确答案是 **A 多项式的度(polynomial degree)**。

在多项式回归中，多项式的度直接决定了模型的**复杂度（Capacity）**，是调节欠拟合与过拟合之间平衡（即偏差-方差权衡）的最核心超参数。

### 为什么选择 A？

多项式回归的数学表达式如下：
$$y = w_0 + w_1x + w_2x^2 + \dots + w_dx^d + \epsilon$$

其中 $d$ 就是多项式的度。
*   **欠拟合 (Underfitting)：** 当 $d$ 选得过小（例如 $d=1$，即线性回归）时，模型过于简单，无法捕捉数据中复杂的非线性真实规律。此时模型具有**高偏差 (High Bias)**。
*   **过拟合 (Overfitting)：** 当 $d$ 选得过大时，模型变得极其灵活，不仅学习了数据中的真实信号，还把随机噪声 $\epsilon$ 当作规律死记硬背了下来。这会导致训练误差极低，但在测试集上表现极差。此时模型具有**高方差 (High Variance)**。

<summary><b>点击展开：为什么其他选项不是主要影响因素？</b></summary>

*   **B 是否通过矩阵求逆/梯度下降学习权重：** 
    这仅仅是**优化算法（Optimization）**的差异。无论是使用解析解（如正规方程进行矩阵求逆）还是使用数值解（梯度下降），它们的目标都是去最小化同一个损失函数（如均方误差）。只要算法能够正确收敛，找到的最优权重 $w$ 是基本一致的，**不会改变模型本身的复杂度**，因此不影响过拟合或欠拟合的状态。
*   **C 高斯噪声方差 (variance of the Gaussian noise)：** 
    噪声是数据集中固有的、不可预测的随机误差（即不可约误差）。虽然数据中噪声方差大时，高次多项式更容易去拟合这些噪声（从而放大过拟合的负面影响），但噪声方差是你无法控制的客观数据属性，它本身**不作为调节模型拟合状态的参数**。
*   **D 每一次训练的输入个数固定：** 
    这通常指的是训练时的**批量大小 (Batch Size)**。批量大小主要影响梯度下降时的**收敛速度和参数更新的稳定性**。虽然极其小的 Batch Size 会引入一些正则化效果，但它对模型最终是欠拟合还是过拟合的影响，远远不及模型自身的复杂度（多项式的度）来得根本和巨大。
