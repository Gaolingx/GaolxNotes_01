# 双星号传参打包解包

python传参加**有什么含义？

### 🤖 Assistant

在Python中，在函数传参时使用 `**`（双星号）主要有两种截然不同但又密切相关的含义。它的具体作用取决于它是用在**函数定义**时，还是**函数调用**时。

简单来说，`**` 扮演着**“打包”**和**“解包”**字典的角色。

以下是详细解释：

### 1. 用在“函数定义”时：收集关键字参数（打包成字典）

当你在定义函数时使用 `**`（通常习惯命名为 `**kwargs`，即 keyword arguments），它的含义是：**“把所有未被明确定义的关键字参数，收集起来打包成一个字典 (dictionary)。”**

这使得你的函数可以接受任意数量的关键字参数，极大地提高了代码的灵活性。

**代码示例：**

```python
def print_user_info(name, **kwargs):
    print(f"姓名: {name}")
    print(f"其他信息 (字典形式): {kwargs}")
  
    # 可以遍历这个字典
    for key, value in kwargs.items():
        print(f"  {key}: {value}")

# 调用函数：传入了一个明确的参数 name，和三个额外的关键字参数
print_user_info(name="Alice", age=25, city="Beijing", job="Engineer")
```

**输出结果：**
```text
姓名: Alice
其他信息 (字典形式): {'age': 25, 'city': 'Beijing', 'job': 'Engineer'}
  age: 25
  city: Beijing
  job: Engineer
```

### 2. 用在“函数调用”时：解包字典（展开为关键字参数）

当你在调用一个已经定义好的函数时使用 `**`，它的含义是：**“把一个字典解包，将其中的键值对转换为独立的关键字参数传递给函数。”**

字典的键（key）必须是字符串，且必须与函数定义中的参数名相匹配。

**代码示例：**

```python
def create_profile(username, email, age):
    print(f"用户: {username}, 邮箱: {email}, 年龄: {age}")

# 我们有一个字典，里面包含了需要传给函数的数据
user_data = {
    "username": "Bob",
    "email": "bob@example.com",
    "age": 30
}

# 使用 ** 进行解包调用
create_profile(**user_data) 

# 上面这行代码等价于：
# create_profile(username="Bob", email="bob@example.com", age=30)
```

### 3. 常见的高级用法：参数透传

在编写装饰器（Decorators）或者类的继承（如 `super().__init__(**kwargs)`）时，经常会把打包和解包结合起来使用。这被称为参数透传（Forwarding arguments），意味着一个函数可以原封不动地把接收到的关键字参数传递给内部的另一个函数。

```python
def target_function(a, b, c):
    print(f"a={a}, b={b}, c={c}")

def wrapper_function(**kwargs):
    print("在调用目标函数前做一些事情...")
    # 接收任意关键字参数（打包），然后再传给目标函数（解包）
    target_function(**kwargs)

wrapper_function(a=1, b=2, c=3)
```

<summary><strong>💡 扩展阅读：与单星号 `*` 的区别</strong></summary>

为了更全面地理解，这里简单对比一下单星号 `*`：

*   **单星号 `*` (如 `*args`)**：处理的是**位置参数 (Positional arguments)**。
    *   定义时：将多余的位置参数打包成一个**元组 (tuple)**。
    *   调用时：将列表或元组解包成独立的多个位置参数。
*   **双星号 `**` (如 `**kwargs`)**：处理的是**关键字参数 (Keyword arguments)**。
    *   定义时：将多余的关键字参数打包成一个**字典 (dictionary)**。
    *   调用时：将字典解包成独立的多个关键字参数。

**标准顺序：**
如果在同一个函数中同时使用，必须遵循严格的顺序：`常规参数` -> `*args` -> `**kwargs`。

```python
def my_func(a, b, *args, **kwargs):
    pass
```
