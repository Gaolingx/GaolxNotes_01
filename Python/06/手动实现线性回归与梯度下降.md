对于线性回归，我们的目标是拟合一个线性模型：
$$ \hat{y} = XW + b $$

我们使用均方误差（MSE, Mean Squared Error）作为损失函数：
$$ Loss = \frac{1}{N} \sum_{i=1}^{N} (\hat{y}_i - y_i)^2 $$
其中 $N$ 是样本数量，$\hat{y}$ 是预测值，$y$ 是真实值。

为了使用梯度下降优化 $W$ 和 $b$，我们需要求损失函数对它们的偏导数（即反向传播计算梯度）：
1. 损失函数对预测值的导数：
$$ \frac{\partial Loss}{\partial \hat{y}} = \frac{2}{N} (\hat{y} - y) $$
2. 对权重 $W$ 的导数（链式法则）：
$$ dW = \frac{\partial Loss}{\partial W} = X^T \cdot \frac{\partial Loss}{\partial \hat{y}} = \frac{2}{N} X^T (\hat{y} - y) $$
3. 对偏置 $b$ 的导数：
$$ db = \frac{\partial Loss}{\partial b} = \sum \frac{\partial Loss}{\partial \hat{y}} = \frac{2}{N} \sum (\hat{y} - y) $$

最后，使用学习率 $\alpha$ 更新参数：
$$ W = W - \alpha \cdot dW $$
$$ b = b - \alpha \cdot db $$

### 完整代码实现

你可以直接复制以下代码并在你的 Python 环境（如 Jupyter Notebook 或直接运行 `.py` 脚本）中运行。它会自动生成模拟数据、进行训练，并弹出图表显示拟合结果和 Loss 下降曲线。

```python
import numpy as np
import matplotlib.pyplot as plt

# ==========================================
# 1. 生成模拟数据
# ==========================================
np.random.seed(42) # 固定随机种子以保证结果可复现

# 生成 100 个样本的特征 X (从 0 到 2)
X = 2 * np.random.rand(100, 1)

# 设置真实的权重和偏置
true_W = 3.0
true_b = 4.0

# 生成对应的真实标签 Y，并加入一些高斯噪声
# 真实公式: y = 3x + 4 + noise
y = true_W * X + true_b + np.random.randn(100, 1) * 0.5

# ==========================================
# 2. 初始化模型参数和超参数
# ==========================================
# 随机初始化权重 W 和 偏置 b
W = np.random.randn(1, 1)
b = np.random.randn(1)

learning_rate = 0.1 # 学习率
epochs = 50         # 迭代次数
N = len(X)          # 样本数量

losses = []         # 用于记录每次迭代的损失

# ==========================================
# 3. 训练循环 (手动实现 Forward, Backward, Optim)
# ==========================================
for epoch in range(epochs):
    # --- 前向传播 (Forward Propagation) ---
    y_pred = np.dot(X, W) + b
  
    # 计算损失 (MSE Loss)
    loss = np.mean((y_pred - y) ** 2)
    losses.append(loss)
  
    # --- 反向传播 (Backward Propagation) ---
    # 计算梯度
    error = y_pred - y
    dW = (2/N) * np.dot(X.T, error) # 对 W 的梯度
    db = (2/N) * np.sum(error)      # 对 b 的梯度
  
    # --- 梯度下降更新参数 (Gradient Descent) ---
    W = W - learning_rate * dW
    b = b - learning_rate * db
  
    # 每 10 次迭代打印一次日志
    if (epoch + 1) % 10 == 0:
        print(f"Epoch {epoch+1}/{epochs} - Loss: {loss:.4f} - W: {W[0][0]:.4f} - b: {b[0]:.4f}")

print("-" * 30)
print(f"训练完成！\n真实参数: W={true_W}, b={true_b}")
print(f"预测参数: W={W[0][0]:.4f}, b={b[0]:.4f}")

# ==========================================
# 4. 可视化结果
# ==========================================
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))

# 图1：数据散点图与拟合直线
ax1.scatter(X, y, color='blue', label='Actual Data', alpha=0.6)
ax1.plot(X, np.dot(X, W) + b, color='red', linewidth=2, label='Fitted Line')
ax1.set_title('Linear Regression Fit')
ax1.set_xlabel('X')
ax1.set_ylabel('y')
ax1.legend()
ax1.grid(True)

# 图2：Loss 下降曲线
ax2.plot(range(1, epochs + 1), losses, color='green', marker='o', markersize=4)
ax2.set_title('Loss (MSE) Reduction over Epochs')
ax2.set_xlabel('Epochs')
ax2.set_ylabel('Loss')
ax2.grid(True)

plt.tight_layout()
plt.show()
```

### 代码原理解析

1. **`y_pred = np.dot(X, W) + b`**：这是线性回归的核心公式，利用矩阵乘法一次性计算所有样本的预测值，这比使用 Python 的 `for` 循环要高效得多。
2. **计算梯度 `dW` 和 `db`**：
   * `dW` 中使用了 `np.dot(X.T, error)`，这一步不仅利用矩阵乘法实现了误差与特征的相乘，同时隐式地完成了所有样本梯度的求和运算（公式中的 $\sum$）。
   * `db` 只需要对所有样本的误差进行求和，所以直接使用 `np.sum(error)`。
3. **Loss 可视化**：代码右侧的图表会将 `losses` 列表中的数据画出来。你会明显看到一条类似于反比例函数的绿色曲线，Loss 在前几个 Epoch 急剧下降，随后逐渐平缓收敛，这说明我们的梯度下降算法成功找到了最优解。
